MVP notes

This app is education and planning software, not investment, tax, or legal advice.

  • MVP scope: U.S. Treasury marketable securities only.
  • Included types: Bills, Notes, Bonds, TIPS, FRNs, and CMBs.
  • Data is loaded from a same-origin GitHub Pages JSON snapshot refreshed by GitHub Actions.
  • Direct Treasury marketable securities are modeled in $100 face-value increments.
